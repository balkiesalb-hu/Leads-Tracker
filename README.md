# 🔖 Leads Tracker - Chrome Extension

A simple yet powerful Chrome extension to save and manage your important links and current tab URLs.

## ✨ Features

- **Save Manual Input** - Enter any URL or text and save it to your list
- **Save Current Tab** - One-click save of the active tab's URL
- **Persistent Storage** - Your leads are automatically saved using localStorage
- **Delete All** - Double-click to clear all saved leads
- **Open Links** - Click any saved lead to open it in a new tab

### Manual Installation (Developer Mode)
1. Clone this repository or download the ZIP
   ```bash
   git clone [https://github.com/balkiesalb-hu/leads-tracker]
2. Open Chrome and navigate to chrome://extensions/
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked"
5. Select the extension folder

The extension icon should appear in your toolbar

## 🎯 How to Use
**Add a lead manually:**
- Type a URL or text in the input field
- Click the "SAVE INPUT" button
  
**Save current tab:**
- Navigate to any website
- Click the extension icon
- Click the "SAVE TAB" button

**Delete all leads:**
- Double-click the "DELETE ALL" button
  
**Open a lead:**
- Click any saved link to open it in a new tab
